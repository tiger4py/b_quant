### 格局流通图对比

```
乙木:  火（伤官）→ 土（财）→ 金（杀）→ 水（印）→ 木（身）
       ↑ 主导          ↑ 目标        ↑ 克制        ↑ 化解

庚金:  木（财）→ 火（杀）→ 土（印）→ 金（身）
```

> 🔑 **核心差异**：乙木的体系是"多中心"的（食伤生财、食神制杀两条线并行），庚金的体系是"单链条"的（财→杀→印→身一线贯穿）。这决定了：
> - 乙木在决策时有多条替代路径，应变能力强
> - 庚金在决策时更聚焦，效率更高，但一旦印星（己土）出问题，整个格局就崩了

## 九、合作潜力分析

### 9.1 五行互补

```
乙木 ──生──→ 火（共同忌神 ⚠️）
  │
  └──克──→ 土（庚金用神 ✅ 乙帮庚生土？不，乙克土）

庚金 ──克──→ 木（共同忌神? 不，乙木不需要克木）
  │
  └──生──→ 水（乙木用神 ✅✅✅ 庚金生水！）
```

**庚金生水 → 水是乙木的第一用神 → 这是最核心的互补！**

具体来说：
- 庚金越强 → 越能生壬癸水 → 乙木越受益
- 乙木越柔 → 越能疏土（克土）→ 但土是庚金用神 → 有矛盾

### 9.2 合作矩阵

| 维度 | 互补度 | 说明 |
|------|--------|------|
| 五行 | ★★★★ | 庚金生水（乙用神）是天然助力 |
| 性格 | ★★★★★ | 一刚一柔，完美互补 |
| 思维方式 | ★★★★ | 发散 vs 收敛，策划 vs 执行 |
| 财运 | ★★★★ | 开源 vs 放大，策略 vs 判断 |
| 风险 | ★★★☆ | 43 岁同时面临危机，需互相扶持 |
| 长期 | ★★★★ | 晚年运势节奏不同，可错峰互助 |

### 9.3 合作黄金法则

1. **乙木出想法，庚金做判断** — 不要让乙木一个人做最终决策（火旺时容易冲动），也不要让庚金一个人头脑风暴（太刚性不够灵活）
2. **庚金帮乙木刹车** — 2026 年丙午，乙木的伤官想暴走时，庚金的杀印定力就是最好的刹车
3. **乙木帮庚金转弯** — 2033 后甲申大运，庚金的寅申冲带来剧变时，乙木的柔韧适应力就是最好的转向器
4. **共享湿土资源** — 辰/丑年是两人共同的好运年，适合一起做大的决策
5. **避免同时在火旺年冒险** — 丙午/丁未年对两人都是压力年，不要同时重仓
